Key New features

  • Hardware resolver support (RESOLVER)
  • Smart placement (Smart Placement)
  • OptiFlash Support (OptiFlash Memory Technology)
  • Memory Configurator (Memory Configurator)
  • Uniflash tool support (TI Uniflash Tool)
  • OSPI NAND flash support without DMA (OSPI Nand Flash IO)
  • Trigonometric Math Unit (TMU) support (TMU Operations and Benchmarking)

Modules Not tested/supported in this release

  • Ethernet phy (ETHPHY) board driver
  • ICSS-EMAC
